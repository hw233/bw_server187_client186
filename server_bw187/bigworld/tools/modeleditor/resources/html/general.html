Menu Shortcuts

Key Action Menu Item
Ctrl+A Adds selected model to the viewport.
(Useful for editing parts of a SuperModel) 		File→Add Model
Ctrl+B Regenerates model's visibility box using its animations. File→Regenerate Visibility Box
Ctrl+H Toggles visibility of all panels. View→Show Panels
Ctrl+O Opens the Open dialog. File→Open Model
Ctrl+R Discards changes and reloads original model. File→Revert To Saved
Ctrl+S Saves current model. File→Save
Ctrl+T Reloads texture files applied to model.
(Useful to refresh modified textures) 		File→Reload All Textures
Ctrl+Y Redoes the most recent undone model-changing action. Edit→Redo
Ctrl+Z Undoes the most recent model-changing action.
The mouse cursor must be over the viewport for shortcut to be enabled.
Only actions that change the model (such as modifying model's material or adding/removing animations and LODs) are added to the redo list.
Actions like changing any field in Display tab, or changing camera's angle or height are NOT added to the list. 		Edit→Undo
