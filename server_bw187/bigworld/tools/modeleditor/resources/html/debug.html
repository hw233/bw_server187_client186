Debug (Watchers) Console (activated by F7)

When Selecting:

Key Action
Backspace Returns to tree's previous level.
Enter If selected node is not leaf:

Moves one level down in tree.

If selected node is leaf:

Edits selected watcher.

Home Returns to tree's root.
Page Down Scrolls the list down.
Page Up Scrolls the list up.
Numpad +
(Plus sign) 		Scrolls watcher list down.
Numpad -
(Minus sign) 		Scrolls watcher list up.
Shift+Enter Displays modification history.
0-9 Selects item 0-9.
Shift+(0-9) Selects item 10-19.
Ctrl+(0-9) Selects item 20-29.
Ctrl+Shift+(0-9) Selects item 30-39.
Alt+(0-9) Selects item 40-49.
Alt+Shift+(0-9) Selects item 50-59.

When Editing:

Key Action
(Changes watcher value by:)
Page Up +1
Page Down -1
Alt+Page Up +1,000
Alt+Page Down -1,000
Ctrl+Page Up +100
Ctrl+Page Down -100
Shift+Page Up +10
Shift+Page Down -10
