Movement Control

Key Action
Caps Lock Increases camera speed.
A Moves camera to the left.
D Moves camera to the right.
E Moves camera downwards.
J Sets the control to the joystick.
K Sets the control to the keyboard.
Q Moves camera upwards.
S Moves camera backwards.
W Moves camera forward.
Ctrl+1 Sets camera speed to Slow.
Same as Slow Camera toolbar button ().
Ctrl+2 Sets camera speed to Medium.
Same as Medium Speed Camera toolbar button ().
Ctrl+3 Sets camera speed to Fast.
Same as Fast Camera toolbar button ().
Ctrl+4 Sets camera speed to Super Fast.
Same as Super Fast Camera toolbar button ().
