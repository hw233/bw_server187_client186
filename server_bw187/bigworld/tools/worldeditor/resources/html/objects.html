Object Mode

Key Action
Alt Activates the Scale gizmo.
Shift Activates the Rotation gizmo.
Del Deletes selected objects.
Enter Places object under the mouse cursor.
Esc Deselects the selected objects.
C Clones the selected objects.
Mouse left
button click 		Selects object.
Mouse left
button drag 		Selects group of objects.
Mouse scroll
wheel 		Rotates selected objects.
