Editing

Key Action
F4 Cycles through object locking modes:
 • Free
 • Terrain
 • Obstacle
Same as Objects tab's Locking Mode field.
F8 Cycles through coordinate filters:
 • World
 • Local
 • View
~ (Tilde) Toggles player walktrough mode.
Same as Player Preview Mode () toolbar button.
Also, same as Options tab's, Player Preview Mode check box.
B Toggles BSP drawing mode.
Same as setting Options tab's, Show BSP field to Custom.
R Recreates chunk.
M Toggles drag-on-select mode.
Same as Objects tab's Drag On Select check box.
V Makes a copy of selected object(s) before moving by dragging.
Ctrl+G Toggles object grid snap mode.
Same as Objects tab's Object Grid Snaps group, Enabled check box.
Ctrl+L Cycles through lighting modes:
 • Standard
 • Dynamic
 • Specular
Shift+F4 Captures copy of current view to a local bitmap file.
Mouse middle
button click/L		 Links selected waypoint to waypoint under the cursor.
