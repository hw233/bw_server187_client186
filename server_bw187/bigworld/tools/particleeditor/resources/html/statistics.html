Realtime Profiling Console (activated by F5)

Key Action
F5 Toggles Realtime Profiling console.
Enter Expands/collapses selected. element
Esc Exits Realtime Profiling console.
Home Scrolls the list to display first page.
Page Down Moves selection to next element in the list.
Page Up Moves selection to previous element in the list.
Num Pad +
(Plus sign) 		Scrolls the screen one line down.
Num Pad -
(Minus sign) 		Scrolls the screen one line up.
[
(Left square
bracket) 		Moves selection to previous element in the list.
]
(Right square
bracket) 		Moves selection to next element in the list.
G Draws a graph for the selected element.
