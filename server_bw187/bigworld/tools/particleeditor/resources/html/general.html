Menu Shortcuts

Key Action Menu Item
Ctrl+H Toggles visibility of all panels. View→Show Panels
Ctrl+O Opens the Open dialog. File→Open Particle Directory
Ctrl+S Saves current particle system. File→Save Particle System
Ctrl+T Reloads textures used by the particle system, and that were modified by a third-party application since it was loaded. Edit→Redo
Ctrl+Y Redoes the most recent undone action. Edit→Redo
Ctrl+Z Undoes the most recent action. Edit→Undo
